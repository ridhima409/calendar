# Calendar UI (React + TypeScript + Tailwind CSS)

This is a minimal Vite + React + TypeScript project that demonstrates a simple Calendar UI component.

## Setup

1. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

2. Start dev server:
   ```bash
   npm run dev
   ```

3. Open the URL printed by Vite (usually http://localhost:5173).

Notes:
- Tailwind is configured via tailwind.config.cjs and PostCSS.
- The calendar component is in `src/components/Calendar.tsx`.
